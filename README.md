# 智能家居环境监测系统

基于 STM32F103 微控制器的智能家居环境监测系统，实时监测温湿度、人体移动、有害气体浓度，并通过 OLED 显示屏展示数据，异常情况触发蜂鸣器报警。

## 项目简介

本项目是一个嵌入式智能家居监测系统，主要用于室内环境监测和安全预警。系统通过多种传感器采集环境数据，实现以下功能：

- **温湿度监测**：实时检测室内温度和湿度
- **人体感应**：检测是否有人体移动
- **气体检测**：监测一氧化碳等有害气体浓度
- **数据显示**：OLED 屏幕实时显示各项数据
- **安全报警**：异常情况自动触发蜂鸣器报警

## 硬件平台

| 组件 | 型号/规格 |
|------|----------|
| 主控芯片 | STM32F103C8T6 |
| 温湿度传感器 | DHT11 |
| 人体红外传感器 | HC-SR501 |
| 气体传感器 | MQ-7 |
| 显示屏 | OLED (SSD1306) |
| 报警器 | 蜂鸣器 |

## 环境要求

### 开发工具

- **IDE**: Keil MDK-ARM 5.x
- **调试器**: ST-Link V2
- **固件库**: STM32F1xx HAL Driver

- STM32CubeMX (可选，用于配置外设)
-串口调试助手 (可选，用于调试输出)

3. 烧录完成后，按复位键运行程序

## 功能模块说明

### DHT11 温湿度传感器

- 引脚：PA8
- 温度测量范围：0-50°C
- 湿度测量范围：20%-90%RH
- 采样周期：≥1秒

### HC-SR501 人体红外传感器

- 检测方式：被动红外感应
- 检测距离：3-7米（可调）
- 输出：数字信号（有人/无人）

### MQ-7 气体传感器

- 检测气体：一氧化碳 (CO)
- 测量范围：10-1000ppm
- 采集方式：ADC 模拟采集

### OLED 显示屏

- 分辨率：128x64
- 接口：I2C
- 显示内容：温度、湿度、人体状态、气体浓度

### 蜂鸣器报警

- 触发条件：
  - 检测到人体移动
  - 气体浓度超过阈值

## 注意事项

1. DHT11 传感器采样间隔需 ≥1 秒，否则可能导致读取失败
2. MQ-7 传感器首次使用需要预热 24-48 小时以达到最佳精度
3. HC-SR501 传感器上电后需要约 1 分钟初始化时间
4. 建议使用 3.3V 供电，确保系统稳定性
